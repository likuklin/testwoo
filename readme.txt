== External services ==
The plugin provides the possibility of ordering shipment(s) within the ppl.cz shipping service. Data about the delivery address, the person who is to receive the shipment, his/her email and phone number for contacting him/her in case of a problem with the delivery of the ordered shipment are shared. The basic url is https://api.dhl.com/ecs/ppl/
